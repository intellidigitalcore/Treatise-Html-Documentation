overview

Treatise - HTML Template

Treatise is an eye-catching HTML5/CSS theme luxury design for personal bloggers. With enough theme options to give you plenty of control without being overwhelming with beautiful blog layouts, single post layouts, Post adding in via Rich Text editor and optional plugins available to give your theme an extra boost. it’s super simple and easy to setup,fast and a breeze to use. You will get high quality responsive, well crafted blog out of the box to make writers only focuses on writing content. it has great typography to make your fans and followers focus on every word you write.


It’s built with the most modern tools (HTML5, Tailwind, ES6 and JQuery) . it’s a very complete and highly customizable template that will help you build your own website layouts. It’s Simple, elegant, extensive, professional.

Change Site Title and Favicon

Go to every Html file (ex: index.html) to change the title and fav icon in <header> tag.

Change Logo

Go to every Html file (ex: index.html) and change logo in <header> tag.

Change Loader

Go to every Html file (ex: index.html) and change Loader in just upper of <header> tag.

Customize Menu

Go to every Html file (ex: index.html) and change menu items in <header> tag.

Customize Footer

Go to every Html file (ex: index.html) and change footer logo, social media, footer data in <footer> tag.

Change Copyright Infomartaion

Go to every Html file (ex: index.html) and change copyright information in <footer> tag.

Section Data

Go to every Html file (ex: index.html) and here you will find every section in <div> tag with well comment.

Blog Posts

Go to src/AllBlogPage.html file. Here you will fild all the all blog container. Or you can search by the classname "all-posts" to find this.

Single Blog Posts

Go to src/SingleBlog.html file. Here you will find the blog details Having simple layout well comment.

Create Blog

Go to src/createBlog.html to find all the create-blog code. Or you can find by the classname "create-blog" to find this section.

Author Profile

Go to AuthorProfile file (src/AuthorProfile.html) and to changes regarding Author

Go to src/AuthorProfile.html file to get author profile settings search by classname "profile-edit-options".

Membership Prices

src/membership.html file and you can change the price or other info for membership.

Typography - Font Family

Go to /main.css file and put your font family link to change the font of the website.

Typography - Color

Go to /tailwind.config.js file and put your color code to change font and other colors.
